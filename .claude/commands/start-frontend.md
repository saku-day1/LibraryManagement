Vue 3 + TypeScript の開発サーバーを起動してください。

## 手順

1. `frontend/` ディレクトリが存在するか確認する
   - 存在しない場合は「Phase 1 がまだ実装されていません」と伝えて終了する
2. 依存パッケージが未インストールの場合（`node_modules/` がない場合）は先に `npm install` を実行する
3. 以下のコマンドで開発サーバーを起動する

```bash
cd frontend && npm run dev
```

## 起動後に確認すること

- `http://localhost:5173` でアクセスできることを伝える
- ポート 5173 がすでに使用中の場合は Vite が自動で別ポートを割り当てるため、ターミナルの出力に表示される URL を案内する
